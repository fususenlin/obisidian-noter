# Noter 快速参考

## 🚀 快速开始

### 安装
```bash
cd /Users/limaosheng/codebase/noter
./install.sh /path/to/your/obsidian/vault
```

### 打开 Noter
- 点击 Obsidian 右侧边栏的 📝 图标
- 或按 `Ctrl/Cmd + P`，输入 "Noter"

---

## 📝 使用笔记

### 创建笔记
1. 在输入框中输入内容
2. 使用 `#标签` 语法（如：`#灵感 #工作`）
3. 按 `Ctrl/Cmd + Enter` 或点击"发送"

### 查看笔记
- 点击卡片展开/收起
- 滚动浏览所有笔记
- 按时间倒序显示

### 删除笔记
- 点击卡片展开
- 点击"删除"按钮

---

## 🏷️ 标签管理

### 添加标签
- 输入时使用 `#标签名`
- 自动识别并高亮

### 筛选标签
- 点击标签云中的标签
- 点击 "#全部" 取消筛选

### 搜索标签
- 在搜索栏中输入标签名
- 实时显示结果

---

## 📁 存储格式

### 文件位置
```
Vault/Noter/Notes/YYYY/MM/YYYYMMDD.md
```

### 示例
```
Vault/Noter/Notes/2026/03/20260301.md
```

### 笔记格式
```markdown
[ID] {标签 1，标签 2} [ISO 时间] 笔记内容

---
[下一条笔记]
```

## 💡 最佳实践

### 标签命名
- 简洁明了（如：`#工作` 而非 `#工作相关的事情`）
- 保持一致性
- 使用层级标签（如：`#项目/Noter`）

### 笔记内容
- 一条笔记一个想法
- 保持简洁
- 善用标签分类

### 定期回顾
- 每天花 5 分钟回顾
- 每周整理标签
- 每月清理无用笔记

---

## 🐛 故障排查

### 插件不显示
1. 检查文件是否复制到正确位置
2. 重启 Obsidian
3. 查看控制台错误

### 笔记不保存
1. 检查文件夹权限
2. 确认路径配置正确
3. 查看控制台错误

### 样式不正常
1. 确认 `styles.css` 已复制
2. 清除 Obsidian 缓存
3. 重启 Obsidian
